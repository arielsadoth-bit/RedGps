Opcion 2: URL fija con Render

Este proyecto ya esta preparado para Render con:
- Dockerfile
- render.yaml
- DATA_DIR=/var/data para guardar la base SQLite en disco persistente

Pasos:
1. Sube este proyecto a GitHub.
2. Entra a https://render.com
3. Crea una cuenta o inicia sesion.
4. New + > Blueprint.
5. Conecta tu repositorio de GitHub.
6. Selecciona este proyecto.
7. Render leera render.yaml y creara el servicio redgps-examen.
8. Cuando termine el deploy, te dara una URL fija parecida a:
   https://redgps-examen.onrender.com

Importante:
- La base SQLite se guarda en /var/data dentro del disco persistente.
- No cierres nada en tu computadora: Render corre el sistema en internet.
- Ya no necesitas Cloudflare ni abrir-publico.bat para esta opcion.
